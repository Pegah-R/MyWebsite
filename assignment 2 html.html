Assignment 2: Parametrics and Generation

This assignment is a series of 21 sketches, which were done every day for three weeks. This practice of daily sketching is a way to gradually build our skills in working with p5.js, which in the end add up to be significant. These daily sketches do not all have to be completely different they could be improved or changed versions of each other that were done the day or days before.

I've had a bit of experience working with javascript in high school, which helped me quite a lot with understating more complicated codes.

As a class, we all agreed on a statement, which had 26 letters. Each letter was then assigned to each person and the sketches were done based on that letter, and I was assigned “N”. The letters are then put together to make a poster, and using the Risograph method to print the poster in one tone or one color.

The phrase agreed by the class: I see trees; green, red roses too.
